# Fake News Detection using Machine Learning
## Project Description:
This project uses machine learning techniques to classify news articles as **Fake** or **Real**.  
Natural Language Processing (NLP) is used to process the text and train a classification model.
## Features
- Text preprocessing and cleaning
- Feature extraction using NLP techniques
- Machine learning model for classification
- Predicts whether a news article is fake or real
## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- Natural Language Processing (NLP)
